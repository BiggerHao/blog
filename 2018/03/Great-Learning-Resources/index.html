学习资料汇总

发布于：2018-03-29

我不爱管理浏览器的书签，导致很多保存的资料都没有整理起来，所以，以后我会把我认为不错的学习资料整理在这篇博客中。

课程类

  1. CS 294: Deep Reinforcement Learning, Fall 2017 from Berkeley: 在 YouTube 上 还有课程视频，该播放列表只能通过链接访问。

  2. Pattern recognition / Speech processing from TAMU.

  3. Approximation Algorithms and Hardness of Approximation from EPFL. (不知道为什么这个网页会占用很多CPU，建议不要长时间打开这个网页。)

  4. CS261: Optimization and Algorithmic Paradigms from Stanford.

  5. Probability and Computing, 2016-17 from Oxford.

视频类

  1. Essence of linear algebra from 3Blue1Brown: 用动画讲解线性代数中的关键概念，可以让人非常直观地理解一些概念的几何意义。

书籍类

  1. Neural Networks and Deep Learning by Michael Nielsen: 讲解神经网络和深度学习，这本书的内容就公开在这个网站上。
  2. Python Data Science Handbook by Jake VanderPlas: 介绍了用 Python 处理数据常用的一些库，在 GitHub 上 有其 Jupyter notebook 版本。

杂项类

  1. Tutorials on topics in machine learning: 分类汇总了 1995 年到 2006 年计算机领域会议中的 Tutorial，适合在刚接触一个领域时阅读。

  2. 一些不错的配色方案网站，可以用于图表制作中。

    • Curated color palette inspiration
    • Color Hunt
    • Colors - House of can Schneider
