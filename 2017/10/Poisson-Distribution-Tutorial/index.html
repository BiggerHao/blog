泊松分布

2017-10-02

泊松分布、均匀分布、指数分布之间有什么关系呢？

泊松分布与其他分布的关系

对于一个泊松过程，如果在时间区间 0 到 \(t\) 之间有且仅有一个事件发生，那么这个事件的发生时间在 0 到 \(t\) 内是均匀分布。这个事件的总发生次数 \(N(t)\) 是一个泊松随机变量。事件发生一次后再次发生的等待时间则服从一个指数分布。
