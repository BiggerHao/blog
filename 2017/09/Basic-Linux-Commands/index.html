vim 编辑器

我们进入 test 文件夹，使用 vim 编辑器新建一个文件 MIT。

此时我们处于编辑器的一般模式，仅可以上下翻阅文件。按下键盘的 I 键，进入编辑模式，就可以对文件内容进行编辑。将如下内容复制到该文件中，然后按 esc 键退出编辑模式，再依次按 :wq 三个键，然后按下回车即完成对文件内容的保存并退出编辑器。 在一般模式下，按下 : 键后，我们实际上是进入了编辑器的命令行模式。

如果对文件编辑有误，不想保存文件内容，可以将上述步骤中最后的 :wq 换为 :q! 这三个按键，即可强制退出编辑器，并且不保存当前修改。

vim 编辑器还有一些常用的快捷键。在一般模式下，按 ^ 可以将光标定位至行首，按 $ 可以将光标定位至行尾，连续按两下 d 可以删除当前行。按 : 进入命令行模式后，可以输入一个数字将光标定位至指定行。除了可以按 : 从一般模式进入命令行模式外，还可以通过按 / 进入命令行模式，再输出一个字符串，可以进行搜索。一般模式下，依次按下 /of，可以搜索文件中的所有 of 的匹配，通过按 N 键，可以将光标定位至下一个匹配处。
